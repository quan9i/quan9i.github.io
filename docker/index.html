kali环境安装docker搭建漏洞平台

发表于 2022-01-26 | 分类于 docker |
字数统计: 488 | 阅读时长 ≈ 1


我会努力追上曾经那个寄予厚望的自己

近期需要在搭建一个本地环境，用来扩充命令执行注入漏洞等相关知识，所以我选择在虚拟机中用kali操作系统安装docker来搭建靶场，
记录安装过程如下：
首先需打开终端，安装docker，语句如下
apt-get install docker.io

发现报错，说权限不够，需要root权限，此时我们输入如下语句
sudo su -

将我们之前设置的密码输入
注:密码是不显现的，直接输入即可，不要以为是自己电脑卡了的问题

输入后可以看到已经得到root权限了
此时再次输入安装docker的语句
apt-get install docker.io

如图所示，已经可以安装（到最后会出现一个选择框，分为yes和no，选yes）
安装完成后，需要再安装一下docker-compose，语句如下
pip install docker-compose

发现缺少pip3模块，百度过后了解到了一个冷知识
kali自带有python2和python3，但是没有安装pip2和pip3
因此我们需要自己安装一下pip3模块，语句如下
apt-get install python3-pip

到最后会出现一个框，直接按enter键进行
安装完成后再次安装docker-compose,语句如下
pip install docker-compose

安装完成后，执行下一步，下载vulstudy项目，语句如下
git clone https://github.com/c0ny1/vulstudy.git

然后这里需要安装的基本就完成了，现在就可以选择我们用的漏洞平台了，我用的是DVWA漏洞平台，那就跳转到该目录下，语句如下
cd vulstudy/DVWA

开启容器和关闭容器的语句如下
docker-compose up -d #启动容器
docker-compose stop #停止容器

完成后即说明搭建完毕
查看ip地址，输入如下语句
ip-addr

访问http://192.168.134.132/

任务结束，晚安。

参考文章：https://cloud.tencent.com/developer/article/1586659
