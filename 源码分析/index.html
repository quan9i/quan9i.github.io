函数扩充

1、check函数
CHECK 约束用于限制列中的值的范围。
示例：
check_input($value)表示的是只接受输入是变量value的值
实例：

2、ctype_digit()函数
作用：用来检测一个字符串中所以的字符是否都为纯数字
示例：
ctype_digit($text);
如果变量text是纯数字，就输出true，否则就输出false
3、isset（）函数
isset() 函数用于检测变量是否已设置并且非 NULL
菜鸟讲解示例：

4、get_magic_quotes_gpc函数
当 magic_quotes_gpc 打开时，所有的 ‘ (单引号), “ (双引号), (反斜线) and 空字符会自动转为含有反斜线的溢出字符
5、stripslashes函数
删除反斜杠
6、php ctype_digit() 函数
ctype_digit()：检测字符串中的字符是否都为纯数组
示例：
ctype_digit($text);
如果 $text 中的所有字符都为纯数字，则返回 TRUE 否则返回 false
7、mysql_real_escape_string() 函数转义 SQL 语句中使用的字符串中的特殊字符。
下列字符受影响：
\x00
\n
\r

‘
“
\x1a
如果成功，则该函数返回被转义的字符串。如果失败，则返回 false。
8、intval（）函数
intval() 函数用于获取变量的整数值。
intval() 函数通过使用指定的进制 base 转换（默认是十进制），返回变量 var 的 integer 数值。 intval() 不能用于 object，否则会产生 E_NOTICE 错误并返回 1。
语法
int intval ( mixed $var [, int $base = 10 ] )
参数说明：

$var：要转换成 integer 的数量值。
$base：转化所使用的进制。

如果 base 是 0，通过检测 var 的格式来决定使用的进制：

如果字符串包括了 "0x" (或 "0X") 的前缀，使用 16 进制 (hex)；否则，
如果字符串以 "0" 开始，使用 8 进制(octal)；否则，
将使用 10 进制 (decimal)。
