python request模块安装及所遇问题小记

发表于 2022-01-19 | 分类于 python |
字数统计: 586 | 阅读时长 ≈ 2


总有一片晚霞会落在你头上，总有一天你也会快乐一场。

Python可使用第三方库requests发送请求，需要先安装requests

查看是否有pip

Python安装模块使用pip命令，现在高版本Python在安装时已经自动安装了pip，不需要再安装，在python安装目录的Scripts下
对于Windows用户，打开cmd，切换到scripts目录下，操作为cd D:\python\Scripts，具体见下图

安装requests

输入pip install requests，如下图

此时可见下方出现警告，翻译一下可以知道是由于pip版本过低引起的问题，解决方案为
在cmd中输入python -m pip install --upgrade pip -i https://pypi.douban.com/simple，具体如下图

此时输入pip list查看当前版本，正常的可能这时候就已经更新好了，但我这个却又报了个该死的警告，如下图

此时我也很麻木呀，无奈之下，继续百度，终于，找到第二个解决方案，输入代码python -m pip install --upgrade pip -i https://pypi.douban.com/simple(python是国外的玩意，下载比较慢，搞一个镜像可以下载快很多)，如下图

此时再输入pip list查看pip版本，如下图

果然，功夫不负有心人，成功了！
此时继续安装我们的requests模块，输入pip install requesets，但好运又一次降临，这次从黄色警告升级为了红色错误，如下图

那这个时候怎么办呢，那就屁颠屁颠的去找度娘吧
百度后发现解决方案python -m pip install --upgrade pip，如下图

此时再输入pip3 install requests，注：pip与pip3没有什么大区别，只是当一台电脑同时有多个版本的Python的时候，用pip3就可以自动区分用Python3来安装库。是为了避免和Python2发生冲突的。
具体如下图

此时显示成功安装，我们可以再进行进一步检测，输入python进入python，再输入import requesets，若此时无报错则说明安装成功，如下图

安装成功，没想到安装个requests模块都能有这么多问题，我这个小废物搞了十几分钟才搞好，还是需要加油啊，晚安！
