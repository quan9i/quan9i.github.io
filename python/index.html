运算符优先顺序


前三个为算术运算符，挨着的三个为位运算符，再后面的一个是比较运算符，其后两个是布尔运算符，最后一个是赋值运算符
因此常规情况下，优先顺序为：
算术运算符>位运算符>比较运算符>布尔运算符>赋值运算符
为什么说常规，因为但当存在小括号时，优先计算小括号内内容
